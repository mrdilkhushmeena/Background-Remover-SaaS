# Background Remover SaaS API

## 🚀 Overview
A SaaS platform that removes image backgrounds and provides a secure API for developers.

## 🔑 Features
- Admin-controlled API keys
- Image background removal
- API usage logging
- Dashboard for API management

## 🛠 Tech Stack
- Flask
- SQLite
- Pillow
- rembg

## 📡 API Usage

POST /api/remove

Headers:
X-API-KEY: your_api_key

Body:
file: image
